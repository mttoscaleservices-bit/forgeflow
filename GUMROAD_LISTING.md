## Title
Next.js 16 SaaS Starter Kit – Stripe + Auth + Dashboard + PostgreSQL

## Description

**Launch your SaaS in hours, not weeks.**

This production-ready starter kit gives you everything you need to build and sell a subscription-based SaaS product. Built with Next.js 16, Stripe, PostgreSQL, and TypeScript.

### What's Included

- **Landing Page** – Beautiful hero section with gradient design, animations, and testimonials
- **Authentication** – Register/Login with NextAuth v5 (Credentials provider)
- **Protected Dashboard** – Full responsive dashboard with sidebar navigation
- **Pricing Page** – Monthly/Annual toggle with 20% annual discount
- **Stripe Payments** – Checkout, subscriptions, webhooks, and Customer Portal
- **3 Pricing Tiers** – Free, Pro ($29/mo), Enterprise ($99/mo)
- **Customer Portal** – Users can cancel, upgrade, or update payment method
- **PostgreSQL Database** – Prisma ORM with auto-detection (SQLite for local dev)
- **Auto Database Detection** – Works seamlessly in local and production
- **SEO Optimized** – Open Graph, Twitter cards, and meta tags included
- **Dark Mode Ready** – Built with Tailwind CSS dark mode support
- **Python Tools Included** – Lead generator, price monitor, image optimizer, SaaS manager CLI

### Tech Stack

| Technology | Purpose |

### How to Launch

1. Clone the repo
2. Set your environment variables
3. Run `npm run dev`
4. Deploy to Vercel with one click
